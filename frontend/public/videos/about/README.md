## Video Specs

- **Format**: MP4 (H.264)
- **Resolution**: 1920×1080 (Full HD)
- **Duration**: 10-20 seconds per clip
- **Frame Rate**: 30fps
- **Bitrate**: 5-8 Mbps recommended

## Fallback Behavior

- If MP4 is missing → poster image displayed (static)
- If poster is missing → gradient background fallback
- On slow connections (2G/slow-2G) → poster only (no video)
- `prefers-reduced-motion` → poster only
