# About Experience — Video Assets

Drop your generated Veo MP4s here. The code will pick them up automatically.

## Expected Files

| Filename | Section | Description |
|----------|---------|-------------|
| `hero.mp4` | Hero | Hexagonal tunnel fly-through |
| `problem.mp4` | Problem | Globe with threat hotspots |
| `architecture.mp4` | Architecture | 3-layer glass platform tilt-up |
| `ml-models.mp4` | ML Models | Neural network zoom-in |
| `competitive.mp4` | Competitive | Server rack → hexagonal node dolly |
| `tech-stack.mp4` | Tech Stack | Orbital shot around dark cube |
| `cta-portal.mp4` | CTA | Iris door opening to command center |
| `poster-*.webp` | All | Static poster frames (1 per video) |

## How to Generate

See **[`docs/VEO_VIDEO_GENERATION_GUIDE.md`](../../docs/VEO_VIDEO_GENERATION_GUIDE.md)** for:
- Exact copy-paste Veo prompts
- ffmpeg optimization commands
- Poster frame extraction
- File size targets

## Target Sizes

- Desktop MP4: **< 3 MB** each
- Mobile MP4: **< 1 MB** each (optional `*-mobile.mp4` variants)
- Poster WebP: **< 100 KB** each
