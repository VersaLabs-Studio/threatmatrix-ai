## Bug #1: PCAP Upload FK Violation

**File:** `backend/app/services/pcap_processor.py`  
**Location:** `create_upload_record()` method, line ~510-530  
**Root cause:** `uploaded_by` is set to a DEV_MODE mock UUID (`00000000-...0001`) that doesn't exist in the `users` table, violating the FK constraint.

**Fix:** Wrap `uploaded_by` with a NULL fallback — if the user_id doesn't exist in the DB, pass `None` instead. Add a DB check before insert.

```python
async def create_upload_record(self, filename, file_size, file_path, user_id=None):
    upload_id = str(uuid4())
    now = datetime.now(timezone.utc)

    # Verify user exists in DB to avoid FK violation
    verified_user_id = None
    if user_id:
        async with async_session() as session:
            result = await session.execute(
                text("SELECT 1 FROM users WHERE id = :uid"),
                {"uid": user_id}
            )
            if result.scalar():
                verified_user_id = user_id

    async with async_session() as session:
        await session.execute(
            text("""
                INSERT INTO pcap_uploads (
                    id, filename, file_size, file_path, status,
                    uploaded_by, created_at, updated_at
                ) VALUES (
                    :id, :filename, :file_size, :file_path, 'processing',
                    :uploaded_by, :now, :now
                )
            """),
            {
                "id": upload_id, "filename": filename,
                "file_size": file_size, "file_path": file_path,
                "uploaded_by": verified_user_id,  # None in DEV_MODE
                "now": now,
            },
        )
        await session.commit()
    return upload_id
```

## Bug #2: `EXCLUDED.now` SQL Error

**File:** `scripts/populate_ml_models.py`  
**Location:** Line 149  
**Root cause:** `EXCLUDED.now` references a bind parameter `:now`, not a column. PostgreSQL `ON CONFLICT DO UPDATE` only supports `EXCLUDED.<column_name>`.

**Fix:** Change line 149 from:
```sql
updated_at = EXCLUDED.now
```
to:
```sql
updated_at = CURRENT_TIMESTAMP
```
